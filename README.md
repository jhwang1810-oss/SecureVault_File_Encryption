# 🔐 SecureVault CLI

A lightweight command-line encryption tool using AES-GCM and PBKDF2 for secure message storage.

---

## 🚀 Features
- AES-GCM encryption (confidentiality + integrity)
- Password-based key derivation (PBKDF2)
- Secure file storage
- Tamper detection
- Simple CLI interface

---
